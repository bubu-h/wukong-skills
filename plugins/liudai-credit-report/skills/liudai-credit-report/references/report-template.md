# 流动资金贷款授信审查意见模板

> 本模板基于实际授信审查意见材料整理，适用于单户流贷授信审查报告。

---

## 一、授信申报情况

**开头第一句话必须是以下格式（核心要素须齐全，其余内容按需补充）：**

> **XX分行申请XX公司综合授信总额度XXXX万元，其中，综合授信敞口额度XXXX万元，授信有效期X年，授信品种为：XXXX（品种为必填项；串用规则、单笔期限、保证金比例等按需补充）。担保方式为XXXX。**

**示例：**
> 南京分行申请中国东方航空江苏有限公司综合授信总额度100000万元，其中，综合授信敞口额度100000万元，授信有效期一年，授信品种为：短期流动资金贷款、银行承兑汇票、非融资性保函（履约保函、预付款保函、质量保函）、国内信用证及商业承兑汇票贴现（含保贴）额度100000万元，额度之间可串用，单笔业务期限不超过1年，银票、国内证及保函均免保证金。担保方式为信用。

**各要素说明：**
- **分行名称**：申报分行全称
- **客户名称**：授信主体全称
- **授信总额度**：综合授信总额度（含低风险额度）
- **敞口额度**：综合授信敞口额度（不含保证金覆盖部分）
- **授信期限**：如"一年""两年"
- **授信品种**：**必填**。列出所有品种，如短期流动资金贷款、银行承兑汇票、国内信用证、非融资性保函、商业承兑汇票贴现（含保贴）、反向保理、国际贸易融资等
- **串用规则**：各品种间是否可串用（如"各品种间可串用""不可串用"）
- **单笔业务期限**：如"单笔业务期限不超过1年"
- **保证金比例**：各品种对应的保证金要求（如"银票保证金不低于20%""国内证免保证金"）
- **担保方式**：信用/保证/抵押/质押，以及具体担保人/抵质押物概述

**后续补充内容（按以下规则处理）：**

- **集团统一管理**：
  - 若材料中有明确说明：写"纳入XX集团统一管理"
  - 若材料中未找到相关内容：写"==请关注所属集团=="

## 二、授信主体简述和业务简述

本节只写三项内容：

1. **客户性质**：央企、地方国企、民企、外资、混合所有制等（从材料中提取）
2. **申报性质**（**需加黄色底纹**）：
   - **纯新增**：第三节"存量授信及用信情况"中写明"本次申报纯新增业务，无存量授用信"
   - **存量续作**：第三节有上轮授信内容，当前在授信有效期内，且上轮授信敞口金额与第一节申报的授信敞口金额一致
   - **存量增额**：第三节有上轮授信内容，当前在授信有效期内，且上轮授信敞口金额小于第一节申报的授信敞口金额
   - **存量减额**：第三节有上轮授信内容，当前在授信有效期内，且上轮授信敞口金额大于第一节申报的授信敞口金额
   - 写法示例："==本次申报纯新增授信10亿元=="、"==本次申请存量续作周转类授信0.5亿元=="、"==本次申请存量增额周转类授信0.5亿元=="、"==本次申请存量减额周转类授信0.5亿元=="
3. **审批权限级别**（**需加黄色底纹**）：
   - 统一写为：本次业务审批权限为总行级/部门级审批会
   - **由用户确认具体级别**，skill 不自动判断。写作示例：==本次业务审批权限为总行级审批会==。

**范例：**

> 借款人为**央企下属子公司**（客户性质）。==本次申报纯新增周转类授信10亿元==（申报性质，黄色底纹）。==本次业务审批权限为总行级/部门级审批会==（审批权限，黄色底纹）。

## 三、存量授信及用信情况

**处理规则：**

- **此前无授信（纯新增）**：
  - 统一写：==**本次申报纯新增业务，无存量授用信。**==
  - 整句标黄，方便用户回头自行核对（用 `==` 包裹）

- **此前有授信**：
  - **按用户提供的材料原文来写，不要做任何修改**
  - 材料中关于上轮批复、当前用信的内容是什么，就原样写什么
  - 包括但不限于：批复编号、核定额度、授信品种、担保方式、前提条件、当前用信余额及明细等

## 四、授信业务合规性审查

**固定格式：**

> 本笔业务流程合规、材料合规、授信主体符合我行授信授信政策。==**审批权限为总行级/部门级审批会。**==

## 五、授信主体信用风险审查

### （一）非财务因素分析

**1. 基本情况**

必须包含以下要素（**整合为一个自然段，不得逐条列示**）：
- 成立时间
- 注册资本或实收资本
- 注册地址
- 法定代表人
- 股权结构（股东构成、持股比例）
- 经营范围

可根据用户提供的材料在段落中添加其他合适的内容。

写法示例：
> 借款人成立于2015年3月18日，注册资本8000万元人民币，注册地址为江苏省镇江市丹徒区瑞业路9号，法定代表人张建国。股权结构方面，张建国持股75%（实际控制人），李明持股25%。经营范围包括汽车零部件、精密机械配件、铝合金铸件的研发、生产与销售。

> 以上信息如有材料不足，在缺失项补充阶段要求用户提供。

**2. 经营情况**

必须介绍清楚：
- **主营业务是什么**：具体产品/服务、业务模式
- **主营产品/服务结构**：各产品/服务占比
- **产能、产量、产能利用率、产销率（仅限生产类企业）**
- **主要上下游客户分析**（附客户明细表）
  - 上游：主要供应商、采购内容、结算方式、账期
  - 下游：主要客户、销售内容、结算方式、账期
- **收入构成及变化趋势**
- **经营变化情况**：近年经营策略调整、业务板块变化、产能扩张/收缩等
- **行业地位及竞争优势**
- **环保及资质情况**（如有）

> 以上信息如有材料不足，在缺失项补充阶段要求用户提供。

### （二）财务因素分析

> **核心原则：行业针对性分析**
>
> 财务报表数据分析必须结合客户所在行业和主营业务特征，有针对性的选取分析重点，不能机械套模板。不同行业关注的财务科目和指标完全不同，分析要体现行业专业性。
>
> **行业分析指引（举例，不限于此）：**
> - **贸易/流通行业**：重点关注存货周转率、应收账款周转率、经营性现金流与营收的匹配度、毛利率稳定性。对存货要区分在途/在库，应收账款要关注集中度和账龄。
> - **制造/生产行业**：重点关注固定资产规模与产能匹配度、折旧政策、存货结构（原材料/半成品/产成品）、毛利率变动与成本传导能力、产能利用率对固定成本摊薄的影响。
> - **建筑/工程行业**：重点关注应收账款（含已完工未结算）、存货中的工程施工、应付账款规模与账期、经营性现金流与营收的背离程度、垫资模式下的资金压力。
> - **IT/软件行业**：重点关注无形资产（研发资本化）、人员规模的费用化影响、应收账款回款周期、毛利率水平（通常较高但需分析可持续性）。
> - **房地产行业**：重点关注存货（开发成本/开发产品）、预收账款/合同负债、有息负债规模与期限结构、现金流与开发周期的匹配。
>
> 面对具体客户时，根据其行业属性选取对应的分析重点，而非机械套用下面的通用框架。对于与行业特征明显不符的数据要重点分析原因。

**1. 财务报表数据分析**

- **审计情况**：
  - 审计年度及会计师事务所
  - 审计意见类型（标准无保留/保留/无法表示意见等）
  - 即期报表是否经审计

- **简要财务报表（表格列示）**：
  - 用表格列示近三年及一期的简要财务数据
  - 表格包含：资产负债表关键科目、利润表关键科目、现金流量表关键科目
  - 示例表格结构（**根据客户行业特征调整具体科目**，以下为制造类企业参考示例）：

    | 项目 | 20XX年 | 20XX年 | 20XX年 | 20XX年X月 |
    |------|--------|--------|--------|-----------|
    | 货币资金 | | | | |
    | 应收账款 | | | | |
    | 其他应收款 | | | | |
    | 存货 | | | | |
    | 流动资产合计 | | | | |
    | 固定资产 | | | | |
    | 总资产 | | | | |
    | 短期借款 | | | | |
    | 应付账款 | | | | |
    | 总负债 | | | | |
    | 所有者权益 | | | | |
    | 营业收入 | | | | |
    | 营业成本 | | | | |
    | 税金及附加 | | | | |
    | 销售费用 | | | | |
    | 管理费用 | | | | |
    | 财务费用 | | | | |
    | 净利润 | | | | |
    | 经营活动现金流净额 | | | | |

- **重点科目或变化非常大的科目分析**：

  本 skill 采用"**尽力而为 + 异常必报**"原则：

  **（1）尽力而为**：根据已有材料，能分析到什么程度就到什么程度。
    - 材料数据充足 → 分析变动原因和趋势，结合行业特征做针对性判断
    - 材料数据不足 → 写到哪算哪，不强求深度（如只有余额无账龄，就只写余额和增减变化）
    - 分析内容包括：变动金额、变动原因、对经营的影响

  **（2）异常必报**：若某科目显著不符常理，但已有材料无法解释原因时：
    - 在报告中用 `==标黄==` 标注该科目及数据，附注"因材料未提供XX明细，暂无法分析原因"
    - **同时**，在给用户的缺失清单中，点名指出该异常项，并告知用户在报告的哪个位置做了标注
    - 典型异常情形举例：
      - 应收账款占总资产比例过高（如>50%）但无账龄结构和客户明细
      - 营收大幅增长（如>30%）但成本几乎不变，或成本大增但营收不变
      - 资产负债率显著偏离行业平均水平
      - 经营现金流与净利润长期背离（如连续盈利但经营现金流为负）

  **通用参考科目**（**根据客户行业筛选和调整**）：
    - 货币资金（含保证金、非受限资金明细）
    - 应收账款（前五大客户明细及变动原因，结合行业回款周期判断合理性）
    - 存货（构成及变动原因，结合行业特点分析存货结构是否合理）
    - 固定资产（构成及产能匹配度，制造类企业重点关注）
    - 短期借款（明细及变动）
    - 应付账款（前五大供应商明细）
    - 其他应收款/其他应付款（大额明细及性质）
    - 长期股权投资（明细）

- **关键财务指标**（**根据行业特征选取核心指标，非全部罗列**）：
  - 资产负债率
  - 流动比率、速动比率
  - 毛利率、净利率
  - **行业特定指标**：如存货周转率、应收账款周转率、总资产周转率、利息保障倍数等（根据客户行业选择）

- **趋势分析**：
  - 营收增长趋势及原因
  - 利润变化趋势及原因
  - 现金流质量评价
  - **结合行业周期判断**：趋势变化是行业共性还是企业个性，判断企业经营是在变好还是变差

> 财务数据如有材料不足，在缺失项补充阶段要求用户提供。

**2. 融资及或有负债分析**

**（1）有息负债**

固定文字：
> （1）有息负债：根据分行反馈，客户授用信明细如下所示：

- 在上述文字后**另起一行**，使用**表格**列示客户有息负债情况
- 表格列示内容：融资机构、融资品种、余额、到期日等（根据材料内容调整）
- **如材料中无有息负债明细，向用户反馈缺少该材料**

**（2）征信查询**

固定格式（整段标黄，用 `==` 包裹，提示用户后期自行审核）：
> ==（2）征信查询：根据征信报告显示，借贷交易余额**XX**万元，担保交易余额**XX**万元，对外担保余额**XX**万元。==

- 若用户提供的材料中有征信报告内容：将 XX 替换为实际数字（替换后仍保留 `==` 标记）
- 若用户提供的材料中无征信报告内容：保留 XX，由用户后期补充
- **无论是否填了数字，该整段均标黄**

## 六、授信业务审查

### （一）授信业务用途
- **各场景写法**：
  - 用于借款人自身日常经营周转 → 写"用于借款人日常经营周转"
  - 材料提及可用于子公司但不详 → 写"用于借款人自身或子公司日常经营周转"
  - 材料有明确子公司的具体名称和用途 → 写"用于借款人及子公司XX日常经营周转"
  - 材料提及置换他行借款但不详 → 写"用于置换其他金融机构借款"
  - 材料有明确的被置换机构、金额、原因 → 按材料原文写
- 流动资金需求测算结果（如材料足以支撑测算则列出，如不确定则标注"待测算"）

### （二）第一还款来源
- **默认写法**：第一还款来源为借款人经营业务回款及利润。
- **异常处理**：若分析后认为借款人经营严重亏损、经营现金流持续为负、明显依赖融资续命等情况，上述默认写法**必须加黄色底纹**（用 `==` 包裹），并在其后加括号以斜体注明你的分析意见，例如：
  > ==第一还款来源为借款人经营业务回款及利润。==（*企业连续三年亏损，经营现金流持续为负，实际偿债依赖借新还旧，第一还款来源存疑，需审慎评估*）
- 其他经营性现金流（如有特别说明的事项可补充）

### （三）第二还款来源

先用一小段话总述本次授信的全部担保措施（担保方式、担保方、抵质押物概述）。

然后逐项展开，**每个担保方的阐述控制在 300-800 字**，说重点，避免堆砌无关细节。

- **如为保证担保**：
  - 保证人基本情况（成立时间、注册资本、主营业务、股权结构）
  - 经营情况及主要财务数据
    - **经营描述不能仅罗列数字**，需包含收入构成、主要客户/供应商、经营模式等核心要素
    - 财务数据突出偿债能力关键指标（资产负债率、流动比率、速动比率、经营现金流等）
  - 资信状况
  - **异常提示**：若保证人财务数据有显著异常（如资产负债率>80%、经营现金流持续为负等），在报告中用 `==标黄==` 提示
- **如为抵押担保**：
  - 抵押物权属、地理位置及权证号
  - 建筑面积/土地使用权面积
  - 评估价值、评估日期、抵押率
  - 抵押顺位说明
- **如为质押担保**：
  - 质押物明细
  - 质押率
- **实控人个人担保**：
  - 个人资产情况
  - 个人征信及对外担保情况

## 七、风险点分析

逐条列示风险点，**不要追求数量，把实际存在的风险写全即可**。然后在所有风险点之后，统一列示缓释措施。

> 1. **经营风险**：借款人为汽车零部件制造业务，下游主要为整车厂，对下游议价能力较弱，且当前国内整车厂有一定内卷现象，会在一定程度上压低供应商的利润空间。
>
> 2. **关联交易风险**：企业存在较多关联交易，需关注资金流向，收集相关发票等材料，确保交易价格公允，不存在利益输送和资金挪用。
>
> **缓释措施**：1、借款人整体营收和净利润表现较为稳定，且和多家大型车企有较长的合作。2、集团子公司和实控人提供了担保。3、经营机构需持续关注客户经营情况并审慎用信，一旦发现经营下滑、流动性不足等情况，应及时采取应对措施。

**常见风险类型参考**（实际撰写时根据材料判断哪些确实存在，不存在的不列）：
1. **经营风险**：行业竞争、议价能力、营收下滑、成本上升、产能利用率等
2. **财务风险**：资产负债率、现金流、应收/存货周转等
3. **担保风险**：担保物、保证人代偿能力等
4. **关联交易风险**：资金流向、交易公允性、利益输送等
5. **行业/政策风险**：行业周期、政策调整、环保要求等
6. **挪用风险**：资金挪用、固贷流用等
7. **其他风险**：根据客户实际情况补充

**常见缓释措施参考**：
- 抵/质押担保
- 保证人担保/实控人连带责任担保
- 压缩授信敞口或缩短期限
- 设置用信前提条件
- 加强贷后回检频率
- 要求提供发票等用途凭证

## 八、审查结论意见

**结论**：一句话说明同意/有条件同意/否决。

- **同意/有条件同意**：写明是否对申报方案有修改（如压缩敞口、调整品种、追加担保等），写法参考第一章格式但以"同意"开头。示例：
  > 同意南京分行对中国东方航空江苏有限公司综合授信总额度100000万元，综合授信敞口额度100000万元，授信有效期一年，授信品种为短期流动资金贷款、银行承兑汇票、非融资性保函，担保方式为信用。
- **否决**：写明否决原因，不需要写具体授信方案。

**具体授信方案**（否决时省略）：
- 核定总额度、敞口额度
- 授信期限
- 授信品种及串用规则
- 各品种保证金比例
- 担保方式
- 是否纳入集团管理

**重要前提条件**（如有）：
- 用信上限控制
- 追加担保要求
- 其他前提条件

**重要管理要求**（如有）：
- 贷后回检频率
- 用信监控指标
- 触发降额/退出条件

## 写作规范

### 数据引用规范
- 所有财务数据须注明报告期，如"截至2025年9月末"
- 引用征信报告须注明查询日期
- 引用上轮批复须注明批复编号，如"（BR20250321000003）"
- 金额单位全文统一（万元或亿元）
- 百分比保留两位小数

### 分析深度要求
- **非财务分析**：不仅罗列事实，要有分析判断（如行业地位是否稳固、竞争优势是否可持续）
- **财务分析**：不仅列数据，要揭示变化原因和潜在风险（如应收账款激增说明回款恶化）
- **风险分析**：每个风险点必须有对应缓释措施，不能只列风险不给方案

### 客观性要求
- 分析结论须基于材料中的客观事实
- 对于不确定事项使用审慎措辞（"需关注""一定程度上""存在一定"）
- 不得使用绝对化表述（"必然""绝对""毫无疑问"）
- 材料不足以支撑的内容注明"材料不足，待补充"
